<h1>Tailwindcss - Vite boilerplate</h1>

Vite starter project with tailus ui html setup.

<h2>Tools</h2>

-   Tailwindcss
-   Vite
-   Vanilla
-   Prettier
-   Postcss

## Build Setup

```bash
# install dependencies
$ npm install
# serve with hot reload at localhost:3000
$ npm run dev
# build for production and launch server
$ npm run build
# preview production
$ npm run preview
```
